Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside NFTMAK

All Pages

    • index.html is for Home Page
    • activity.html is for Activity Page
    • category.html is for Category Page
    • collections.html is for Collections Page
    • creators.html is for Creators Page
    • ranking.html is for Ranking Page
    • nft-marketplace.html is for Marketplace Page
    • nft-live-bidding.html is for Live Bidding Page
    • market-single.html is for Market Single Page
    • login-register.html is for Login & Register Page
    • create-item.html is for Item Create Page
    • connect-wallets.html is for Wallets Connect Page
    • author-profile.html is for Author Profile Page
    • blog.html is for Blog standard Page
    • blog-details.html is for Blog details Page
